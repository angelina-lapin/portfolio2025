Project Description

Holidaze is a modern venue booking platform built with React, featuring a manager dashboard, customer booking system, and full API integration. The project allows users to browse available venues, make reservations, and for venue managers to manage their listings and bookings.

Reflections & Improvements

After the initial submission, I improved the project by optimizing API calls, adding user-friendly error handling, and enhancing the responsive design across devices. I also polished the UI with better visual consistency and updated the README with clearer setup instructions based on teacher feedback.
